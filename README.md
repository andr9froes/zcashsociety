# Zcash Society

Privacy as Infrastructure for Self-Governing Communities.

Static site for [zcashsociety.com](https://zcashsociety.com).

## Deploy

Push to GitHub → Vercel auto-deploys from `public/`.

## Structure

```
public/
  index.html    # Single-page manifesto
vercel.json     # Vercel config
```

## Domain

Configure in Vercel → Settings → Domains:
- `zcashsociety.com`
- `zcashsociety.xyz` (redirect to .com)
